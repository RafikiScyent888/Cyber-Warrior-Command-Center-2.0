# Cyber Warrior Command Center

Central landing page for the **Cyber Warrior Program** — a set of interactive
study resources built for students working toward CompTIA A+, Network+,
Security+, and CySA+. This repo hosts the hub page only; each subject area
(sims, acronym games, quiz banks) lives in its own repo and is linked from
here.

Live site: https://rafikiscyent888.github.io/Cyber-Warrior-Command-Center/

## What's here

- `index.html` — the command center itself. Plain HTML/CSS/vanilla JS, no
  build step. Color-coded tiles link out to every resource site, with a
  search bar to filter by keyword. Mobile-friendly.
- `CONCEPTS.md` — a cross-repo concept index: which topics are already
  covered, in which repos, in what format, and where taxonomies or content
  don't yet line up across repos.
- `CLAUDE.md` — full project context (repo inventory, formats, known content
  issues) for anyone (human or AI assistant) picking this project back up.
